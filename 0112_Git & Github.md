## What is Git & Github?



### 1. Git 깃

- (분산) **버전 관리 프로그램**

1. 버전 관리 

- 버전 : 컴퓨터 소프트웨어의 특정 상태

- 관리 : 어떤 일의 사무, 시설이나 물건의 유지, 개량

- 프로그램 : 컴퓨터에서 실행될 때 특정 작업을 수행하는 일련의 명령어들의 모음

- **수정사항, 변경사항을 저장해준다!**

- 백업, 복구, 협업이 가능 -> 수정하는 내용이 다른 사람에게 영향 미치지 않는다

### 2. Github를 이용한 포트폴리오

- 포트폴리오 : 내가 이만큼 관심있고 성실하고 능력있다! 를 보여줄 수 있는 부분

1. Repositories : 프로젝트
2. Star : 즐겨찾기
